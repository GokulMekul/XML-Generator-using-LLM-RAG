# XML-Generator-using-LLM-RAG
Built an LLM-powered system to convert questionnaires into structured XML format. Implemented prompt engineering and text extraction for accurate XML generation. Integrated a RAG pipeline for semantic search across technical manuals. Automated survey scripting workflow, reducing manual XML effort.
